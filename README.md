# Next Word Predictor

## Setup
1. Clone the repo
2. Create conda env: `conda create -n next_word_env python=3.10`
3. Install deps: `pip install tensorflow==2.15.0 keras==2.15.0 streamlit numpy`
4. Download model files from Google Drive (link here) and place in project folder
5. Run: `streamlit run app.py`
